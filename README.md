# Decision_Tree-Bank_loan_Analyzer-
Decision Tree for Bank Loan Approval Prediction
Problem Statement

Financial institutions receive thousands of loan applications every day. Evaluating each application manually can be time-consuming and inconsistent.

This project uses the Decision Tree Machine Learning algorithm to predict whether a customer's loan application is likely to be Approved or Rejected based on multiple financial and personal factors.

Project Objective

Develop a Decision Tree model that analyzes customer information and predicts loan approval decisions using historical data.

Features Used
Age
Annual Income
Credit Score
Loan Amount
Existing Loan
Employment Years

Target Variable:
Loan Status (Approved / Rejected)

Technologies Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Google Colab


Model Evaluation
The model was evaluated using:
Accuracy Score
Confusion Matrix
Decision Tree Visualization
